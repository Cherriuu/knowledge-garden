pip install django # must run everytime you first create a virtual env

django-admin startproject config . # create a new project, gives you configuration files like settings.python3

python3 manage.py migrate # applies database schema changes, run initially to setup table

python3 manage.py runserver # runs a view of your server in port 8000
localhost:8000

python3 manage.py runserver 8001 # run on a different port if other port is in use

python3 manage.py startapp (app_name) # creates folder with all files django needs for an app

After running startapp, remember to register it in config/settings.py under INSTALLED_APPS — otherwise Django doesn't know it exists.
